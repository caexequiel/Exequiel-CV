Curriculum Vitae Web
==================================================================================================
Referencia:
EPITOME es una plantilla de sitio web gratuita para currículum vitae y portafolio personal bellamente diseñada. es moderno
moderno y presenta un diseño visualmente atractivo. Una plantilla de sitio web ideal para creativos
profesionales y autónomos que quieran crear una presencia online que se destaque entre
la media Epitome tiene todos los elementos importantes de un portafolio personal de currículum eficaz
Plantilla de sitio web: un impresionante banner de héroe en pantalla completa, sección acerca de y calificación, servicios,
portafolio, sección de testimonios y contacto. Epitome también es móvil y está preparado para retina.
Se verá genial en cualquier dispositivo, desde móvil hasta escritorio, y en cualquier resolución de pantalla.
